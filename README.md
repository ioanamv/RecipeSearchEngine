# Recipe Search Engine

This is a recipe search web application built with **Elasticsearch** and **ASP.NET MVC**. Users can search recipes by title, ingredients, instructions, cuisine, dietary restrictions, difficulty and cooking time.

## Features

- **Search recipes** using keywords in multiple fields.
- **Display search results** with key information and images.
- **View detailed recipe information** on a separate page.
- **Swagger integration** for API testing.
- **Elasticsearch** with custom mapping for efficient search.
